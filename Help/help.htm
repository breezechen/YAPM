Introduction

Yet Another Process Monitor (YAPM) is a powerful application for Windows NT that allows you to manage the processes and services running on your system.

Requirements

In order to run YAPM correctly, you must have a Windows NT operating system (such as Windows XP or Windows Vista).

Administrator rights are not necessary required, but if you want to take full power on your processes (especially system processes) you should run YAPM as an administrator.

Main screen

Here is a small description of YAPM main screen :

 

 

This is composed of three main areas.

Ribbon

The ribbon regroups all the availables action classified by panel :

-          Processes

-          Services

-          Handles

-          File

-          Jobs

-          Search

-          Misc (miscancellous)

-          Help

Main area

The main area is available for most of the panels (all panels excepted Misc) and display main informations (such as process/service list, search resultsetc.).

Second area

The second area is available only for three panels : Processes, Services and File. It displays some additional informations such as process attributes, service descriptionetc.

For Processes/Services panels, the second area is refreshed when the user clicks on a process in the main area.

 

Processes management

 

 

This is the first panel displayed when you run YAPM. It regroups all actions availables for process management. When the user clicks on a process in the main area, some additional informations are displayed in the second area. You can choose the aditional informations you want to retrieve with the differents checkboxes (1).

-          Retrieve module/threads : retrieve all the loaded modules and the current threads of the process.

-          Retrieve online informations : retrieve the process description online

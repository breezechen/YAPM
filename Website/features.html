List of main features

The main purpose of YAPM is to display all informations availables about processes/services of a local/remote system and all related objects (threads, modules...), and to update all these informations in real time. The application has three principal forms :

  • the main form : displays the list of tasks/processes/services/TCP-UDP connections in real time. The other lists (modules, handles, threads...) can display objects of multiple processes, but are not automatically refreshed
  • the process detail form : displays in real time all informations about a specific process. Double-click on a process to open this form
  • the service detail form : displays in real time all informations about a specific service. Double-click on a service to open this form

For all lists, it is possible to choose the columns to display.

Lots of informations and functions are available, here are the main :

Processes

YAPM is able to kill (simple or kill process tree), stop, resume, set priority, set affinity of processes (multiple selection). You can also choose the informations you want to display (~45 available).

Statistics & performance graphs

YAPM can display a lot of statistics about your processes. YAPM save a complete history of all statistics.

Token

YAPM is able to manage process privileges (you can enable/disable/remove them).

Memory

YAPM can display the memory regions in the virtual memory space of a process. You also can view it in a build-in hexadecimal editor.

Modules

YAPM can unload modules and show complete informations about them. You can of course save a detailed report.

Threads

YAPM can kill, stop, resume and set affinity of threads.

Handles

YAPM is able to list all opened handles (even of system process if your have administrator rights) and show useful informations.

Windows

YAPM can close, set position, show, hide, flash, change caption/opacity, set as active/foreground window...etc. each of opened Windows. You are also able to find the owner process of a specific window by dragging your mouse over the screen.

Services

YAPM can shutdown, stop, resume, pause and change start type of all services.

Monitoring

YAPM allows to see all available Windows perfmon' counters and the evolution of theirs values in a timescale graph. The "log mode" is also a useful feature to know what is done by a process in real time.

Files

YAPM can also manage your files. You can change dates/attributes, find string in file, encrypt, move/copy, delete and even shredd files.

Search

YAPM allows you to search for a specified string into opened processes, modules, handles and windows.

Dependency viewer

YAPM offers the possibility to view dependencies of an executable/dll/...

Remote management

YAPM offers the possibility to manage remote processes of machines of your workgroup via WMI (nothing to install in on the remote machine) or via a client-server architecture (all actions and informations are available, just like if it is a local machine). You can also shutdown, reboot, logoff... remote machines.

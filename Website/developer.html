Introduction

Here is the developer page, you will find some informations about the development of YAPM.

Next release informations

The next version will be released between 08/10/09 and 08/15/09. This will be a major release as a lot of changes have been made : 

  • high optimizations (reduced CPU usage by 50%)
  • lots of bugs fixed
  • no more flickering in the listviews
  • code refactoring (no more VB.Net dlls)
  • works well with UAC under Windows Vista and above
  • etc...

Some new features have been implemented :

  • ability to change memory protection
  • ability to free/decommit memory regions
  • added "highlighting" for processes
  • new informations about processes
  • easy "Copy into clipboard"
  • added an optional statusbar
  • etc...

Progress & bugs to fix

There are about 15 critical bugs to fix.

There are about 80 other changes to do (bug fixes, new options/features...).

Future developments

Future developments will focus on :  

  • Performance++
  • Stability++
  • Make YAPM more powerful for system process management (development of the kernel driver)
  • Make YAPM use less bandwidth for remote monitoring
  • Translation of YAPM in different langages (french in priority ^^)
  • Need a TreeListView to display processes :) I've tried some open source TreeListviews, and the best one seems to be TreeViewAdv.

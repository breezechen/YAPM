How much cost YAPM ?

Nothing. YAPM is a free and open source software, under GNU GPL 3.0 license. But if you want make a donation, you're welcome ! Just click on 'Donate' button in 'Help' view and you will be redirected to the donation page on sourceforge.net.

Why should I use Yet Another Process Monitor ?

YAPM offers lots of useful features that Windows Task Manager does not have. For example, Windows Task Manager can't show loaded modules, threads, windows and handle. You can't search for a specific item, and you can't even select multiple items. If the standard Task Manager is a light and a useful tool, it is really limited.

Process Explorer offers some of these features, but remember Process Explorer is now owned by Microsoft, that's why you can't download its source code. So if you want to add some functions to Process Explorer, you can't ! With YAPM, just ask in the forums and I'll try to implement it (or you can of course do it by yourself if you are able to do it). Moreover,YAPM offers A LOT OF features that Process Explorer does not have (windows, tasks, file, network management, advanced monitoring... etc).

And of course, neither Windows Task Manager nor Process Explorer is able to monitor remote processes :-)

I think I found a bug...

Yes it's possible, as YAPM is always in development. Please report the bugs you found and/or your suggestion to YAPM's tracker.

I can't make the remote connection work...

  • Make sure your firewall is configured to let YAPM work properly
  • Make sure the remote machine is in the same workgroup than the host machine
  • Make sure you have the rights to establish the connection
  • If you are using WMI, make sure WMI in installed on both machine, make sure the RPC server in started and make sure you have entered the good username/password. Normally, you can't connect YAPM to a account which has no password
  • If you are using the client-server method, make sure the IP your entered is correct.
  • If you can't make it work, just ask me !

Introduction

This help file can directly be viewed by selecting 'Help' tab in Yet Another Process Monitor.

Note : depending on the version you are using, the help file may contains differences between your version and the version it describes. The present help file is provided for the last version of YAPM. For now, it is designed for YAPM 2.0, which has not been public released. So that's why you won't find some features described here in your version of YAPM.

Main features

In addition to all basic features available with the standard Windows Task Manager, YAPM offers lots of other useful features :

  • Module and thread management
  • Handle management (keys, files...)
  • View of TCP/UDP connections opened by the processes
  • Windows management (opacity, position, show, hide...)
  • Window' find' process feature : allows to find the process associated to a window via a drag & drop over the screen
